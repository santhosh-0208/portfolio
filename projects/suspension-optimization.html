Suspension System Optimization

Optimization Simulation Control
View on GitHub Download Report (PDF)

Motivation

This project feels very dear to me because this is where it all started.

During COVID, I became fascinated with numerical optimization methods and wanted to apply them to a real engineering problem.

Problem

Designing a suspension involves balancing multiple competing objectives:

  • Ride comfort
  • Body roll
  • Camber variation
  • Stability

The search space is highly coupled and difficult to tune manually. Traditional suspension design relies on intuition and iterative testing, which is both time-consuming and costly.

Approach

I built a comprehensive optimization system consisting of:

  • Double Wishbone Suspension Model - Kinematic and dynamic model of the suspension geometry
  • Dynamic Simulation - High-fidelity simulation of vehicle response to inputs
  • Genetic Algorithm - Built from scratch to explore the design space
  • Objective Function - Combining bounce, roll, and camber metrics

The genetic algorithm evolved population of suspension designs over hundreds of generations, evaluating each design through dynamic simulation and scoring it against the objective function.

Challenges

The biggest challenge wasn't coding the genetic algorithm itself. It was validating the mathematical model.

Many times the code executed perfectly while the physics was completely wrong. Small errors in kinematic calculations or force computations would compound through the simulation, leading to unrealistic results that looked numerically valid on the surface.

I spent considerable time:

  • Validating model predictions against known suspension behaviors
  • Comparing simulation results with real-world data
  • Debugging subtle numerical issues in the kinematics
  • Tuning the objective function weights

Results

The genetic algorithm successfully optimized suspension parameters across all competing objectives. The final designs demonstrated improvements in:

  • Ride comfort with reduced bounce amplitude
  • Better body roll control
  • Improved camber stability across suspension travel

Key Learnings

This project taught me that optimization is only as good as the models and objectives that drive it. A beautiful algorithm is worthless if it's optimizing the wrong thing or working with invalid assumptions.

It also reinforced the importance of validation at every level—from individual equations to complete simulations. The "code runs" is not the same as "code is correct."
